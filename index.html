Le Temps d'Aimer

Le Temps d'Aimer est une plateforme de rencontres dédiée aux personnes de plus de 50 ans, alliant élégance et simplicité.
Chaque section, de l'accueil aux profils, bénéficie d'un design épuré, avec un mode sombre activable pour améliorer l'expérience utilisateur. L'authentification des membres est gérée via un système d'identifiants et de mots de passe simulé en JavaScript, permettant une interaction plus réaliste.

Ce projet met en avant :

  • HTML5 sémantique
  • CSS3 moderne avec variables CSS et dark mode
  • Grid Layout pour la structure
  • JavaScript orienté objets pour la gestion dynamique des profils, des événements et du système de connexion
  • Responsive design pour une expérience utilisateur optimale sur tous les appareils

The Ultimate CS Test

The Ultimate CS Test est une application web interactive de quiz consacrée à l'informatique. Elle propose plusieurs niveaux (Facile, Moyen, Difficile) permettant de tester ses connaissances en HTML, CSS, JavaScript et plus encore.

Chaque question s'affiche avec un effet rétro de machine à écrire animé, et l'utilisateur peut choisir ses réponses dans un style pixel-art avec boutons personnalisés. Un score est généré en fin de partie.

Ce projet met en avant :

  • JavaScript orienté objets (questions, réponses, niveaux, timer)
  • Effets animés personnalisés (typewriter, shuffleText)
  • CSS rétro inspiré des années 90 (pixel art)
  • Navigation dynamique entre les niveaux et feedback utilisateur
  • 100% Responsive

Heart Catcher

Heart Catcher est un mini-jeu développé en JavaScript, mettant en scène un personnage contrôlé par le joueur devant attraper des cœurs tout en évitant des monstres.
Le gameplay est fluide grâce à une boucle `requestAnimationFrame`, et le score est calculé dynamiquement avec gestion du timer, des collisions et du game over.

Ce projet met en avant :

  • Canvas HTML5 pour le rendu du jeu
  • Contrôles clavier en JavaScript natif
  • Animations dynamiques via `requestAnimationFrame`
  • Gestion des collisions et des événements clavier
  • Responsive avec taille de canvas adaptée

Pixel Tic Tac Toe

Pixel Tic Tac Toe est une version moderne et rétro du célèbre jeu du morpion. Le joueur peut choisir de jouer contre un autre joueur réel ou contre une intelligence artificielle avec 3 niveaux de difficulté (facile, moyen, difficile).

Le site intègre un design pixel-art, un mode sombre, un curseur personnalisé ainsi qu’une animation d’introduction pour une ambiance nostalgique.

Ce projet met en avant :

  • JavaScript orienté objets avec logique IA (aléatoire, réflexe et bloquant)
  • Interface responsive avec animations visuelles
  • Choix de l’adversaire en début de partie
  • Mode sombre dynamique avec changement d’image automatique
  • Curseur personnalisé et design rétro inspiré du pixel art

Kitty Click Game

Kitty Click Game est un jeu web rétro où le joueur doit cliquer sur un chat animé le plus de fois possible en 10 secondes.
Le score est sauvegardé localement et un classement dynamique s’affiche à chaque partie. Le bouton "Start" se transforme en compte à rebours interactif, puis en "Restart" à la fin de chaque manche. Un mode sombre et un curseur pixel customisé ajoutent à l’expérience visuelle rétro.

Ce projet met en avant :

  • JavaScript orienté événements et DOM
  • Système de score avec stockage local (`localStorage`)
  • Compteur dynamique dans le bouton
  • CSS customisé avec variables, dark mode et curseur pixel
  • Design responsive avec Grid et Media Queries
